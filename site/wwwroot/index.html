What makes it Gunfight

One life per round

No respawns. Last team standing takes the round, then the killcam rolls.

Shared random loadout

Everyone gets the same primary, secondary, and equipment each round. The rotation rerolls on a timer — pure skill, no class grinding.

First to 6 rounds

Win the round, win the match. Sides switch partway through. Draw rounds don't count toward the limit.

Overtime zone

Tied on a timer expiry? Health decides it — or a capture zone settles overtime when it's close.

No killstreaks, no regen

No streaks, no perks shown pre-round, no health regeneration, no weapon drops. Clean, fair fights.

Auto map scaling

Small lobbies get tight, curated gunfight spawns; 4v4 and up open the whole map automatically.

How a match plays out

  1. 1

    Spawn with the round's loadout

    Both teams get the same weapons. Your loadout slides in on the HUD at the start of each round.

  2. 2

    Wipe the other team

    One life each. Eliminate everyone on the enemy team to win the round before the clock runs out.

  3. 3

    Beat the clock

    If time expires, the team with more total health wins. If it's tied, overtime decides it.

  4. 4

    First to six wins

    Take six rounds to win the match. Sides swap so nobody's stuck on the bad spawn.

Install & join

Plutonium has no automatic mod download — every player installs the mod locally and must run the same version as the server. It takes about a minute.

1

Get Plutonium + BO1

Install the Plutonium launcher and have legitimate Black Ops 1 game files. See the official setup guide.

plutonium.pw setup →
2

Download the mod

Grab the latest mp_gunfight package from the GitHub releases page.

Latest release →
3

Drop it in your mods folder

Extract so the folder lands at:

%LOCALAPPDATA%\Plutonium\storage\t5\mods\mp_gunfight

The folder must stay named mp_gunfight.

4

Load it and join

Launch BO1 multiplayer, open the Mods menu and load mp_gunfight, then find the server in the server browser.

T5 has no direct IP connect — join from the browser list.
